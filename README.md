# PDF 全能工具箱 - 项目总览

## 一句话定位
> 面向淘宝/闲鱼售卖的 Windows 桌面端 PDF 批量处理工具，买断制激活码模式。

## 核心卖点
- 🚀 **批量处理**：一次处理几百个文件，秒杀竞品
- 🎯 **转换质量高**：PDF→Word 格式不乱、不丢字
- 💰 **买断制**：19.9~29.9 元终身使用，性价比碾压订阅制
- 🔒 **离线可用**：不依赖网络，数据不上传，用户放心

## 功能模块一览

| 模块 | 功能 | 优先级 |
|------|------|--------|
| PDF 转换 | PDF→Word / PDF→Excel / PDF→图片 / PDF→HTML | P0 核心 |
| PDF 编辑 | 合并 / 拆分 / 旋转 / 删除页面 / 提取页面 | P0 核心 |
| PDF 压缩 | 无损压缩 / 有损压缩 / 自定义质量 | P0 核心 |
| PDF 安全 | 加密（设置密码）/ 解密（移除密码）/ 权限设置 | P1 重要 |
| PDF 水印 | 添加文字水印 / 图片水印 / 批量去水印 | P1 重要 |
| PDF 提取 | 提取图片 / 提取文字(OCR) / 提取表格 | P1 重要 |
| 图片工具 | 图片转PDF / 白底处理 / 批量改尺寸 / 压缩 | P2 增值 |

## 技术栈速览

| 层 | 选型 | 理由 |
|-----|------|------|
| 语言 | Python 3.11+ | 开发快，PDF生态成熟 |
| GUI框架 | PySide6 (Qt for Python) | 商业友好(LGPL)，控件丰富 |
| 打包 | PyInstaller + Inno Setup | 生成安装包，一键安装 |
| 激活系统 | RSA签名 + 机器码绑定 | 离线验证，防拷贝 |
| 代码保护 | Cython 编译核心模块 | 反编译难度大 |
